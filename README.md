# Traffic-Lights
Simulate Traffic Lights using Arduino Uno
The given code controls traffic lights at cross-roads.
Components required:
-> Arduino Uno
-> Green Leds (4)
-> Red Leds (4)
-> Yellow Leds (4)
-> Jumper wires( as required)
-> USB Cable(to connect your laptop to the Arduino Uno Board)
-> Resistors(no.s 12)
